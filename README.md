# 🐴 Horse Kick: Barrel Smash

Браузерная HTML5-игра с физикой в стиле Angry Birds. Лошадь в ковбойской шляпе пинает задними ногами бочки, которые летят в фермера!

## 🎮 Играть онлайн

👉 **https://bikermiami2-ui.github.io/HorseKick/**

## Управление

| Платформа | Действие |
|-----------|----------|
| 🖥️ ПК | Зажми ЛКМ рядом с лошадью → оттяни назад → отпусти для удара |
| 📱 Мобильные | Тапни и потяни назад → отпусти для удара |

Чем сильнее оттянешь — тем мощнее удар. Пунктирная линия показывает траекторию полёта бочки.

**Цель:** попади бочкой в фермера! 🎯

## Особенности

- 🤠 Лошадь в ковбойской шляпе с процедурной анимацией (дыхание, хвост, удар задними ногами)
- 🛢️ Бочка как снаряд: лошадь пинает реальную бочку, которая летит по параболе
- 👨‍🌾 Фермер-мишень с анимацией получения удара (звёзды из глаз, тряска)
- 🎯 Параболическая траектория при прицеливании
- 💥 Система частиц: щепки, пыль, искры, золотые звёзды
- 🔊 Процедурные звуки через Web Audio API (без внешних файлов)
- ⭐ 5 уровней с нарастающей сложностью
- 🏆 Система звёзд (1-3) за эффективность
- 🐴💬 **«Совет от лошади»** — кнопка с лошадиной мудростью (20 цитат!)
- 💾 Таблица рекордов сохраняется в localStorage
- 📱 Адаптивный дизайн для ПК и мобильных устройств

## Технологии

Чистый HTML + CSS + JavaScript. Никаких движков, сборщиков, npm или CDN. Работает автономно в любом современном браузере.

## Локальный запуск

Просто открой `index.html` в браузере. Сервер не требуется.

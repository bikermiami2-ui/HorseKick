# 🐴 Horse Kick: Barrel Smash

Браузерная HTML5-игра с физикой в стиле Angry Birds. Управляй лошадью, оттягивай прицел и пинай задними ногами бочки!

## 🎮 Играть онлайн

👉 **https://bikermiami2-ui.github.io/HorseKick/**

## Управление

| Платформа | Действие |
|-----------|----------|
| 🖥️ ПК | Зажми ЛКМ рядом с лошадью → оттяни назад → отпусти для удара |
| 📱 Мобильные | Тапни и потяни назад → отпусти для удара |

Чем сильнее оттянешь — тем мощнее удар. Пунктирная линия показывает траекторию полёта.

## Особенности

- 🐎 Процедурная анимация лошади (дыхание, махание хвостом, удар задними ногами)
- 🛢️ Физика бочек: гравитация, столкновения, цепные реакции, разрушение от сильных ударов
- 🎯 Параболическая траектория при прицеливании
- 💥 Система частиц: щепки, пыль, искры
- 🔊 Процедурные звуки через Web Audio API (без внешних файлов)
- ⭐ 5 уровней с нарастающей сложностью
- 🏆 Система звёзд (1-3) за эффективность
- 💾 Таблица рекордов сохраняется в localStorage
- 📱 Адаптивный дизайн для ПК и мобильных устройств

## Технологии

Чистый HTML + CSS + JavaScript. Никаких движков, сборщиков, npm или CDN. Работает автономно в любом современном браузере.

## Локальный запуск

Просто открой `index.html` в браузере. Сервер не требуется.

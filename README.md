# 🐴 Horse Kick: Barrel Smash v2

Браузерная физическая аркада, где лошадь в ковбойской шляпе пинает задними ногами бочки, чтобы попасть в фермера!

## 🎮 [ИГРАТЬ ОНЛАЙН](https://bikermiami2-ui.github.io/HorseKick/)

## Механика
- **Прицеливание** в стиле Angry Birds: оттяни назад и отпусти
- Лошадь пинает **реальную бочку** задними ногами — она летит по параболе
- **Фермер** — мишень, которую нужно сбить
- Бочки сталкиваются друг с другом, разбиваются, запускают цепные реакции
- 5 уровней с нарастающей сложностью

## Управление
| Платформа | Действие |
|-----------|----------|
| 🖥️ ПК | Мышь: кликни рядом с лошадью → оттяни назад → отпусти |
| 📱 Телефон | Тач: тапни рядом с лошадью → потяни назад → отпусти |

## Особенности v2
- 🤠 Ковбойская шляпа на лошади
- 🛢️ Бочка как физический снаряд (не невидимый шар)
- 👨‍🌾 Фермер-мишень с анимацией удара
- 🐴💬 «Совет от лошади» — 20 мудростей в модальном окне
- 💥 Частицы: щепки, пыль, звёзды
- 🔊 Процедурные звуки (Web Audio API)
- 🏆 Таблица рекордов в localStorage
- 📱 Адаптивный дизайн

## Технологии
Чистый HTML5 + Canvas + JavaScript. Никаких зависимостей, сборщиков или CDN.

## Локальный запуск
Просто открой `index.html` в любом браузере.
